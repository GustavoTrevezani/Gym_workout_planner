# Gym Workout Planner

A full stack web application designed to help users create, manage, and track their gym workouts efficiently.

This project is being developed as a **portfolio project** to demonstrate backend and full stack development skills using modern technologies and best practices.

The goal is to simulate the architecture of a **real production system**, including authentication, modular backend structure, and scalable database design.

# 📂 Backend Architecture

The backend follows a layered architecture:

- Routes

- middlewares

- Controllers

- DTO Validation

- Services

- Repositories

- Database (Prisma)

This separation improves maintainability and scalability.
